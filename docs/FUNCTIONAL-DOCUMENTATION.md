# Functional Documentation: Portfolio Module

## Purpose

Provides portfolio tracking and analysis capabilities within the Trading Assistant workspace.

## Key Features

### 1. Watchlist Management

- Create, Read, Update, Delete watchlists.
- Track personalized tickers.

### 2. Portfolio Summary

- View total portfolio value.
- Performance metrics (P&L, Daily Change).

### 3. Market Insights

- Aggregated news and signals relevant to portfolio holdings.

## User Guide

### Accessing the Module

Navigate to `/portfolio` in the main application shell.

### Adding a Ticker

1.  Click "Add Ticker".
2.  Search for symbol (e.g., AAPL).
3.  Select exchange (e.g., NASDAQ).
4.  Confirm.
